# CONTER 点名器

一个跨平台的智能点名器应用，支持多种抽取规则和数据管理。

## 主要功能

- 🎲 多样化抽取规则（随机、权重、最少抽取）
- 📊 Excel 导入导出
- 💾 数据持久化
- 🌍 多语言支持（中文、英文、法语）
- 🎨 Material Design 3 界面

## 系统要求

- Windows 7 SP1 或更高版本
- 50 MB 可用磁盘空间

## 安装说明

运行 CONTER_Setup.exe 并按照向导提示进行安装。

## 数据存储

所有用户数据（学生名单、抽取记录）存储在安装目录的 data 子目录中。
